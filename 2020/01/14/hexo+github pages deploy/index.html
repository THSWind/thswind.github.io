hexo+GitHub pages deploy

1 Preliminary

  1. Window10

2 Install Node.js

You can search ‘node.js’ in browser, and install a stable version.( LTS, i.e.,long term support), then download node. It’s name like this:’node-v8.11.2-x64.msi’.

Install Node.js to a personal disk, (e.g, D disk), and click to ‘add to path’, click to ok.

–testing–

open cmd.exe, then input ‘node -v’. If display the version of node, that means node.js is successfully installed.

3 Sign up for GitHub

When everything is ok, then start a project.

Repository name: your username.github.io (important)

my: thswind.github.io

Tick initialize this repository with a README, then click to Create repository.

–testing–

Create an ‘index.html’ in your username.github.io, and write down:

‘<h1>Hello GitHub Pages</h1>’

Save and submit it, then open ‘username.github.io’ in your browser, if index.html display normally, it means everything is ok.

4 Install git

Search ‘git for windows’ in your browser, download it, it’s name just like: ‘Git-2.17.0-64-bit.exe’. You can install it to the same disk that you have installed Node.js(D disk). When you installing it, remember to click to “Use Windows’ default console window”, done.

–testing–

Open cmd

Input: git

If display some information about it, congratulations，you have successfully installed it.

5 GitHub SHH setting

  1. Open the git-bash.exe

  2. Input:

    ssh-keygen -t rsa -C "your email for sign up for github"

  3. After enter it, the ‘MINGW64’ window will display some information just like:

    Generating publick/private rsa key pair.

    Enter file in which to save the key(/c/Users/Administrator/.ssh/id_rsa):

    It means you can name the file which to save the key, ignore it (recommend), just click to ‘enter’.

  4. After enter it, this window will display:

    Created idrectory '/c/Users/Administrator/.ssh'.

    Enter passphrase (empty for no passphrase):

    Ignore it (recommend), just click to ‘enter’.

  5. Continue to click enter, it will display some information about RSA, it is a key for you log in github.

  6. If previous steps are done successfully, then you can close this window.

  7. Open github, click to your image->Settings->SSH and GPG keys->New SSH key.

    title:'username' (or you can name it randomly)

    Open c:\Users\Adminstrator.ssh, you will find a file, named id_rsa.pub. Copy it’s contents, then paste to Key

    Key: 'id_rsa.pub'

    (tips: If you can’t find id_rsa.pub, you can choose to search ‘id_rsa.pub’ in C disk.)

6 Install Hexo & usage

Search ‘Hexo’ in ur browser, click to “文档|Hexo”, you can see a command for install it. I want to install it to the same disk that installed node.js(D disk).

  1. I create a folder and name it as ‘blog’.

  2. Open cmd, and switch to D disk, then input : cd blog

  3. After you switch to blog folder, just input the command for install hexo, this process will takes lots of time.

  4. Installed it.

  5. Return to your browser, open ‘hexo.io/zh-cn’ page, and roll down to the bottom of this page. There will be several command.

  6. Switch your cmd window,

    D:\blog>

    input ‘hexo init username’,

    It will display INFO Cloning hexo-starter to D:\blog\thswind

    cloning into ‘D:\blog\thswind’

  7. Execute the following command:

    npm install (install some dependency about hexo)

    hexo s -p 5555 (after execute this, you can see your website locally, through input: localhost:5555 in your browser. or ‘hexo s’)

7 Install sublime

  1. Search sublime in ur browser, you can download the latest version. You can also install it in D disk.

  2. Double click sublime_text.exe, and drag the ‘thswind’ folder to the sublime window. It is very convenient.

8 Hexo publish to GitHub

8.1 Website setting

  1. Open ‘_config.yml’(this file is in thswind not the one in themes, remember it you can avoid many problem) in sublime.

    Edit some infomation(disorder):

    deploy:
    type: git
    repo: https://github.com/THSWind/thswind.github.io.git
    branch: master

    url: https://thswind.github.io

    title: THSWind

    author: THSWind

  2. If you want to deploy your post to github, you need to install a deployer.

    D:\blog\thswind>npm install hexo-deployer-git --save

  3. Configure account information

    D:\blog\thswind>git config --global user.email "your email for sign up for github"

    D:\blog\thswind>git config --global user.name "thswind"

  4. Deploy your website

    hexo g to generate website file

    hexo d to deploay website

  5. When you finish step 4, there will be a new window which require you to verify the login information of GitHub account. Just do it!

–testing–

You can Search ‘thswind.github.io’ in your browser. If you can visit your website normally, that means you have finished it.

9 Choose a favorite theme

(The default theme is ‘landscape’, you can through this chapter to change it.)

  1. You can open “文档|Hexo” in your browser again, and click to ‘themes’, then choose a theme which you like.

  2. Git clone [The open source address of the theme]

  3. Copy the theme folder to ‘D:\blog\thswind\themes\[theme]’

  4. Edit the ‘_config.yml’(D:\blog\thswind_config.yml)

    theme: [the name of theme]

That’s all!
