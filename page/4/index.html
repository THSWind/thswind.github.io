import numpy as np

from keras.datasets import mnist

from keras.utils import np_utils

from keras.models import Sequential

from keras.layers import Dense,Dropout,Convolution2D,MaxPooling2D,Flatten

from keras.optimizers import Adam

\# 载入数据

(x_train,y_train),(x_test,y_test)= mnist.load_data()

\# (60000,28,28)

\#(60000,28,28)->(60000,28,28,1) 1是图片的深度，灰度模式

x_train=x_train.reshape(-1,28,28,1)/255.0

x_test=x_test.reshape(-1,28,28,1)/255.0

\# 转化label为 one-hot格式

y_train=np_utils.to_categorical(y_train, num_classes=10)

y_test=np_utils.to_categorical(y_test, num_classes=10)

\# 定义顺序模型

model = Sequential()

\#第一个卷积层

\# input_shape 输入平面

\# filters 卷积核/滤波器个数

\# kernel_size 卷积窗口大小

\# strides 步长

\# padding padding方式 same/valid

\# activation 激活函数

model.add(Convolution2D(

​ input_shape = (28,28,1),

​ filters=32,

​ kernel_size=5,

​ strides=1,

​ padding='same',

​ activation='relu'

))

\#第一个池化层

model.add(MaxPooling2D(

​ pool_size=2,

​ strides=2,

​ padding='same',

))

\#第二个卷积层

model.add(Convolution2D(

​ 64,

​ 5,

​ strides=1,

​ padding='same',

​ activation='relu'

))

\#第二个池化层

tensorflow2.0 notes

2019年1月份，Tensorflow 2.0预览版（alpha)发布，原本1.0版本实现神经网络的方法较多，重复api多。

1 Tensorflow 2.0 特点

  • Tensorflow 2.0清理了废弃的API减少重复和简化了api。
  • 训练方面
    • 使用 Keras 和 eager execution轻松构建模型。为研究提供强大实验工具。
    • Tf.keras :允许创建复杂的拓扑，包括使用残差层，自定义多输入/输出模型以及强制编写的正向传递。轻松创建自定义训练循环。低级Tensorflow API始终可以使用，并可与更高级别的抽象一起工作，以实现完全可定制的逻辑。
  • 部署方面
    • 在任意平台实现稳健的生产环境模型部署
    • 无论在服务器、边缘设备还是在网页上，也不论你使用的是什么语言或平台，Tensorflow 总能让你轻易训练和部署模型。
    • Tensorflow 2.0 通过标准化交换格式来改进跨平台和跨语言部署。

2 课程安排

  • （一）tf.keras

    构建和训练模型的核心高级 API。

    • 单输入单输出 Sequential 顺序模型
    • 函数式 API

  • （二）Eager 模式与自定义训练

    直接迭代和直观调试，Eager模式下求解梯度与自定义训练。

    • Eager 模式 ： 直接迭代和直观调试。
    • tf.GradientTape : 求解梯度，自定义训练逻辑。

  • （三）tf.data

    加载图片数据与结构化数据

  • （四）tf.function

    自动图运算

  • （五）卷积神经网络

  • （六）多输出卷积神经网络综合实例

  • （七）迁移学习

  • （八）模型保存与可视化

  • （九）Tensorboard可视化

encryption+hidden file

1. 文件隐藏

  1. attrib +h +s +r +a [文件全路径]

    通过这种自动添加隐藏属性的方法，将文件变为了操作系统的隐藏文件。提高了安全性。其中的属性可以自己增减。h 表示隐藏属性、 r 只读属性、 a 存档属性（表示该文件在上次备份前已经修改过，对于频繁的文件批量管理很有帮助）、 s 系统文件属性。

2. 信息隐藏

需要dos下的copy命令，将一个文本文件合并到一个非文本文件中，从而实现隐藏的目的。

准备

  1. 一张图片，一个 .txt 后缀的文本文件
  2. copy 图片路径/b + txt文件路径/a 生成路径

举个简单的例子，比如当前我dos定位到两个文件所在的目录时（这里将两个文件放在同一个目录下）。那么采用命令： copy 图片.jpg/b + 文件.txt/a 新文件名.jpg 即可

  1. 此时，如何查看加密信息呢，藏有信息的jpg文件上右键->打开方式->记事本，因为图片是二进制文件，所以上面部分是乱码。我们按住 Ctrl+end 定位到记事本文件的最后，就可以看见刚才加密的信息了。

拓展（copy命令的妙用）

平常我们在图形化界面下复制大文件需要花好多时间，其实如果你懂dos命令，在dos下用命令去复制，那么不管它多大的文件，只需要 2秒 就可以复制完毕。命令格式如下：

copy [路径] [路径]

注意：文件夹复制不了，可以转为压缩包进行复制。尤其比较大的镜像文件效果杠杠的。

3. 音乐加密

以前我曾想象有一种加密方法，是通过音乐来传送密文。这样肯定是出其不意。直到后来我在网上发现了在17-18世纪就有一些数学家和密码学家认为音乐是传递信息最难被破解的方法之一。后来发现有下面几个网站可以做此转换，其算法和加密的方式我并未深入研究。

  6. ctrl+加号： word中把文字转换为下标输入，用ctrl+减号键可以恢复原状
  7. Win10滑动关机： 路径C:\Windows\System32，找到SlideToShutDown.exe，右键发送到桌面快捷方式，双击弹出滑动提示，滑动即可关机
  8. “Win+↑” 最大化 “Win+↓” 最小化
  9. 2、自动关机（命令不带引号）
    ①“shutdown -s -t 1800” （1800秒后关机）
    ②“at 23:00 shutdown -s” （23点关机）
    ③“shutdown -a” （取消关机）
