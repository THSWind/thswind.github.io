CMD命令(其中命令较多，列举最常用的几个)

dir 列举当前目录下所有文件
cd [绝对路径] 进入某路径
cd .. 返回上一级目录

Windows快捷键

命令 功能
Ctrl+shift+n 新建文件夹
alt+d 定位到地址栏
Ctrl+F4 关闭当前文件
Alt+F4 关闭活动项或退出当前应用
Ctrl+Y 还原上一步操作
Ctrl+Z 撤销上一步操作

资源管理器

  1. Ctrl+enter 打开文件夹是在新窗口打开文件夹
  2. Ctrl+shift+n 新建文件夹
  3. Alt+enter 查看文件属性
  4. Alt+左右方向键 资源管理器中进入或退出子目录
  5. Shift+delete 彻底删除

补充

  1. “Win键+数字键（0~9）” ：可以快速启动锁定在任务栏中的前10个对应的程序
  2. 批量重命名文件：在资源管理器中同时选择几个文件，接着按F2键，然后重命名这些文件中的一个，这样所有被选择的文件将会被重命名为新的文件名(在末尾处加上递增的数字)。
  3. 大家都知道F5键可以用来刷新网页，不过它在记事本程序里还有一个作用：一键插入当前的日期和时间。这个看似微不足道的功能有时意外地好用。每写完一条记录或想法，按F5，这就是一个离线的微博哦~ （来自虫部落网友）
  4. 在记事本第一行加入“.LOG”,双引号除外。也可以达到同 3 一样的效果
  5. win10中，多窗口时，按住一个窗口，然后摇动几下，其余窗口会最小化，在继续摇动，会恢复。
  6. ctrl+加号： word中把文字转换为下标输入，用ctrl+减号键可以恢复原状
  7. Win10滑动关机： 路径C:\Windows\System32，找到SlideToShutDown.exe，右键发送到桌面快捷方式，双击弹出滑动提示，滑动即可关机
  8. “Win+↑” 最大化 “Win+↓” 最小化
  9. 2、自动关机（命令不带引号）
    ①“shutdown -s -t 1800” （1800秒后关机）
    ②“at 23:00 shutdown -s” （23点关机）
    ③“shutdown -a” （取消关机）

桌面

命令 功能
Win+A 打开信息栏
Win+D 显示桌面
Win+R 打开运行
Win+E 我的电脑
Win+L 锁定电脑
Win+B 选定桌面右下角工具栏
Win+I 打开电脑设置
Win+home 缩小除当前窗口以外的所有窗口
Ctrl+shift+ESC 打开任务管理器
Win+Ctrl+D 创建新的虚拟桌面
Win+Ctrl+F4 关闭当前虚拟桌面
Win+Ctrl+ →（左右） 切换虚拟桌面
Win + , 临时查看桌面

浏览器

  1. 空格，向下翻页，shift+空格 可以上翻；
  2. Ctrl+h 打开浏览器历史记录
  3. Ctrl+shift+T 恢复刚关闭的网页

Shift+Window键+数字：启动锁定到任务栏中的由该数字所表示位置处的程序的新实例（主键盘区上的数字）
Alt+Windows键+数字：打开锁定到任务栏中的由该数字所表示位置处的程序的跳转列表 （主键盘区上的数字）
Win+X+U+U : win10关机快捷键
ctrl + alt + → :改变屏幕显示方向
