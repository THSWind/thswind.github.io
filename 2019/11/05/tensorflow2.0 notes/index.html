tensorflow2.0 notes

2019年1月份，Tensorflow 2.0预览版（alpha)发布，原本1.0版本实现神经网络的方法较多，重复api多。

1 Tensorflow 2.0 特点

  • Tensorflow 2.0清理了废弃的API减少重复和简化了api。
  • 训练方面
    • 使用 Keras 和 eager execution轻松构建模型。为研究提供强大实验工具。
    • Tf.keras :允许创建复杂的拓扑，包括使用残差层，自定义多输入/输出模型以及强制编写的正向传递。轻松创建自定义训练循环。低级Tensorflow API始终可以使用，并可与更高级别的抽象一起工作，以实现完全可定制的逻辑。
  • 部署方面
    • 在任意平台实现稳健的生产环境模型部署
    • 无论在服务器、边缘设备还是在网页上，也不论你使用的是什么语言或平台，Tensorflow 总能让你轻易训练和部署模型。
    • Tensorflow 2.0 通过标准化交换格式来改进跨平台和跨语言部署。

2 课程安排

  • （一）tf.keras

    构建和训练模型的核心高级 API。

    • 单输入单输出 Sequential 顺序模型
    • 函数式 API

  • （二）Eager 模式与自定义训练

    直接迭代和直观调试，Eager模式下求解梯度与自定义训练。

    • Eager 模式 ： 直接迭代和直观调试。
    • tf.GradientTape : 求解梯度，自定义训练逻辑。

  • （三）tf.data

    加载图片数据与结构化数据

  • （四）tf.function

    自动图运算

  • （五）卷积神经网络

  • （六）多输出卷积神经网络综合实例

  • （七）迁移学习

  • （八）模型保存与可视化

  • （九）Tensorboard可视化

3 Tensorflow 安装

Tensorflow 2.0 兼容的 python 版本：

python 2.7-3.6

环境配置：安装Python 3.6，或者创建一个python3.6的虚拟环境。

安装CPU版本：

pip install tensorflow==2.0.0-alpha0

2019年7月新发布了beta0版本，所以可以安装beta0版本：

pip install tensorflow==2.0.0-beta0

  • 从指定软件源安装多个软件包

    pip install numpy pandas matplotlib sklearn tensorflow==2.0.0-alpha0-i https://pypi.doubanio.com/simple

    上述命令表示从豆瓣软件源安装 numpy，pandas，matplotlib，sklearn，tensorflow2.0.0-alpha-i等软件包。

  • 配合notebook使用，体验更佳

    pip install notebook-i https://pypi.doubanio.com/simple

安装GPU版本

pip install tensorflow-gpu==2.0.0-alpha0

必须在系统中安装 NVIDIA 相关软件（CUDA Toolkit (Tensorflow 支持 CUDA 10.0) 和 cuDNN SDK（7.4.1以及更高版本，用于深度神经网络的GPU加速原语库））
