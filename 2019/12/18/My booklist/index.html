  8. 你还很年轻，将来你会遇到很多人，经历很多事，得到很多，也会失去很多，但无论如何，有两样东西，你绝不能丢弃，一个叫良心，一个叫理想。
  9. 有时候，屈辱地活着，比悲壮地死去更需要勇气。
  10. 从历史中我们可以知道，宽容从来都不是软弱。
  11. 不要以为渺小的，就没有力量；不要以为卑微的，就没有尊严。
  12. 因为你还活着，活着，就要继续前进。
  13. 不知死而死，是谓无知；知死而死，是谓无谓。
  14. 其实在这个世界上，只要你敢忽悠，什么奇迹都是可能发生的。
  15. 得到了权力，似乎就得到了一切，但其实很多人并不明白，在权力的游戏中，你没有休息的机会，一旦参加进来，就必须一直玩下去，直到你失败或是死亡。
  16. 外交，是指处理国与国之间关系的方法，但它还有另外一个通俗的解释—用最礼貌的方式，说出最肮脏的话。
  17. 他只有两样不会，这也不会，那也不会。
  18. 除了好事，什么都干；除了脸，什么都要。
  19. 政治高手就如同江湖大侠，想要金盆洗手一走了之，那是很难的。
  20. 一个人可以影响多数人于暂时，也可以影响少数人于永远，但无法影响多数人于永远。
  21. 纯粹和执着，也是有区别的。所谓执着，就是不见棺材不掉泪，而纯粹，是见了棺材，也不掉泪。
  22. 传说这个世界上存在着一种神奇的东西，它无影无形，却又无处不在，轻若无物，却又重如泰山，如果能够获知这一样东西，就能够了解这个世界上的所有的奥秘，看透所有伪装，通晓所有知识，天下万物皆可归于掌握！
  23. 在清除敌人首脑之前，必须先扫除一切外围和帮手，这就是中华民族的传统智慧，所谓掺沙子、挖墙脚是也。
  24. 历史证明，落水狗如果不打，就会变成恶狼。
  25. 牺牲尊严是不够的，要想在这场残酷的游戏里笑到最后，还必须背离原则，因为眼前的敌手，是一个不讲原则的人。而要战胜一个无原则的对手，唯一的方法就是放弃所有的原则。

2《卑鄙的圣人曹操》

王晓磊[中国]

  1. 何为“霸王之兵”？

    “风林火山，悬权而动，这就是洋洋《孙子》一书最重要的主旨，孟德好眼力。”桥玄忽然站了起来，踱了几步又道，“我也看了半辈子兵法，只有一事未曾参透，何为‘霸王之兵’呢？”

    曹操这会儿真是彻底放开了手脚，朗声道：“霸者，不结成天下诸侯之权也。绝天下之交，夺天下之权，故己威得伸而自私！”他说完这番话，连自己都吓了一跳，这等言语离仁义礼教似乎太远，也忒张狂跋扈了。

  2. 枪杆子里出政权

    场景1-我算看明白了，手里攥着兵，腰杆子就硬。那帮子家伙说是官，其实跟匪也差不多，强占民田、勒索钱粮、结连土豪，杀人就跟碾死臭虫似的。”楼圭侃侃而谈。

    场景2-曹操将宝剑还鞘，不禁怅然道：“本初兄，符节印绶管天下的日子算是到头了，从今以后恐怕要靠手里的刀剑说话了……”

3《睡眠革命》

尼克·利特尔黑尔斯 [英国]

  1. 固定时间起床比固定时间上床更加重要。
  2. 90分钟为一个睡眠周期，看自己是需要几个周期的人，不是每个人都需要睡上八小时。
  3. 两到三点是睡眠高峰期，高效期，尽量不要错过。
  4. 保持口腔湿润，但睡前不要喝太多水。

4《被讨厌的勇气》

岸见一郎[日本]

一个人想要获得幸福，就要实现自我接纳，他者信赖，他者贡献。

5《坏蛋是怎样练成的》

六道[中国]，网络小说

  1. 如果当一个人老了的时候，发现自己一生平平淡淡没有任何好回忆好令自己感动的事，那他的人生注定是失败的。世界上有太多的人一生都是这样的碌碌无为。平平淡淡，如果做坏蛋可以改变这一点的话，那好，我去做，如果我做到了，那我的人生将注定比普通人精彩许多，也刺激许多。
  2. 狼就是不畏惧生死的战士。为了生存，它们会主动攻击比自己强大数倍的敌人，哪怕自己战死，它也要为其他的同类创造出机会。在物竞天择的法则中，狼是可敬的，但也是被人们所不理解、厌恶的。
  3. 流星，因为短暂而美丽，划过黑寂的夜空，释放出那一闪而逝的光芒，虽然微弱，但却没有人能无视它的存在。人生如同流星，充满了精彩与传奇，如同一支美丽的传说，究竟是否会象流星那样短暂，别人决定不了，上天也决定不了，只有自己来掌握。如果愿意，流星也可以变成永恒。
  4. 不要做你想做的事，你一定不是我的对手。
  5. 我非英雄，广目无双，我本坏蛋，无限嚣张。

6《挪威的森林》

村上春树[日本]

  1. 即使你最心爱的人，心中都会有一片你无法到达的森林
  2. 记忆这玩意儿真是不可思议，当我身临其境时，我是一点也不去留意那风景。
  3. 没有人喜欢孤独，只是不想勉强交朋友，要真那么做的话，恐怕只会失望而已。
  4. 我渐渐能意会到，深刻并不等于接近事实。
  5. 当周围一团漆黑的时候,只能静等眼睛习惯黑暗
  6. “喜欢我喜欢到什么程度?”绿子问.”整个世界森林里的老虎全都融化成黄油.”
  7. 火红的晚霞让人置身在火红的绚烂之中， 置身在遥远的童年那天真烂漫的憧憬之中， 置身在那从来未曾实现也永远不可能实现的憧憬之中。

8《鬼谷子》

  1. 摩而恐之，高而动之，微而证之，符而应之，拥而塞之，乱而惑之，是谓计谋。
  2. 无以人之所不欲而强之于人，无以人之不所知而教之于人。人之有好也，学而顺之；人之有恶也，避而讳之。
  3. 天地之变化，在高与深；圣人之制道，在隐与匿。’这讲的就是‘韬晦’二字。
  4. 反以观往，复以验来；反以知古，复以知今；反以知彼，复以知此。动静虚实之理不合于今，反古而求之。事有反而得复者，圣人之意也，不可不察。
  5. 捭阖者，天地之道。捭阖者，以变动阴阳，四时开闭以化万物；纵横、反出、反复、反忤必由此矣。
  6. 因其恶以权之，因其患以斥之。

9《厚黑学》

  1. 笑骂由他人笑骂，好坏我自为之。
  2. 如果认准一个目标，那么就遇事要忍，出手要狠，善后要稳。
  3. 能容忍眼前的侮辱，放下架子，顾全大局，这叫厚。
  4. 读书三决：第一步，以古为敌。第二步，以古为友。第三步，以古为徒。
  5. 一面正义大旗，一个响亮口号，会把天下的人心吸引过来，使原本并不存在正义与否的事业，也变得正义起来。
  6. 古之为英雄豪杰者，不过面厚心黑而已。
  7. 拨开黑的，让人见到真正的。
  8. 讲因果的人，说有个阎王，问阎王在何处，他说在地下。讲耶教的人，说有个上帝， 问上帝在何处，他说在天上。讲理学的人，说有许多圣人，问圣人在何处，他说在古 时。这三种怪物，都是只可意中想象，不能目睹，不能证实。惟其不能证实，他的道 理就越是玄妙，信从的人就越是多。
